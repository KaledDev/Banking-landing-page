# Landing Page de l'Application Bancaire

Cette landing page présente les fonctionnalités de l'application bancaire, permettant aux utilisateurs de découvrir ses avantages et de s'inscrire facilement a la newsletter.

## Fonctionnalités
- Présentation des services bancaires
- Formulaire d'inscription
- Informations de contact

## Technologies Utilisées
- HTML
- TailwindCSS
- JavaScript

## Installation
1. Clonez le dépôt :
    ```bash
    https://github.com/KaledDev/Banking-landing-page.git
    ```
2. Accédez au dossier du projet :
    ```bash
    cd nom-du-projet
    ```
3. Ouvrez le fichier `index.html` dans votre navigateur.
